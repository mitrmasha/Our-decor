# Our-decor
Проект по теме "Наш декор"
## Назначение системы
Система предназначена для автоматизации учета продукции компании, занимающейся производством обоев с использованием различных материалов в качестве основы и покрытия. Система обеспечивает эффективное управление данными о продукции и материалах.

**Поля формы продукции:**
- Артикул
- Тип продукта (выпадающий список)
- Наименование
- Минимальная стоимость для партнера
- Ширина рулона (с точностью до сотых)
